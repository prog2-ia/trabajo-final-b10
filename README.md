# Sistema de Reservas de Pistas Deportivas

## 📖 Propósito del Proyecto
Esta aplicación es un sistema de gestión por consola para un polideportivo. Permite administrar diferentes tipos de pistas, gestionar las tarifas, y organizar un calendario de eventos deportivos, priorizando competiciones oficiales sobre reservas normales mediante la sobrecarga de operadores.

## 🛠️ Instalación y Requisitos
El proyecto está desarrollado en Python puro y no requiere de librerías externas.
Para ejecutarlo, simplemente clona el repositorio:
`git clone [https://github.com/prog2-ia/trabajo-final-b10/blob/main/README.md?plain=1]`

## 🚀 Uso y Ejemplos
Para iniciar el programa, ejecuta el archivo principal desde la terminal:
`python main.py`
